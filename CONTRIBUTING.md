# Contributing

## Code of Conduct
This project adheres to the [Contributor Covenant](https://www.contributor-covenant.org/).

## How to Contribute
1. Fork the repository: `git fork https://github.com/user/multi-agent-orchestrator.git`
2. Create a new branch: `git branch <feature/fix>`
3. Commit changes: `git commit -m '<description>'`
4. Open a pull request: `git push origin <feature/fix>`
